Bitcoin SPV Library

Build apps on top of Bitcoin, Dash or Bitcoin Cash blockchains. This library does the hard work under the hood and enables any capable developer to create trully decentralized apps on top of Bitcoin or some of its supported forks.

Ethereum Light Client Library

Build all kinds of applications on top of Ethereum blockchain that work in a trully decentralized manner.

This library is still in active development phase ... :)
